Frequently Asked Questions

Everything you need to know about hosting OpenClaw in the cloud.

What is OpenClaw (Clawdbot)?

OpenClaw (formerly known as Clawdbot) is an open-source AI assistant that runs in your terminal. Unlike chatbots that just respond to questions, OpenClaw can actually take actions: manage your calendar, send emails, automate tasks, and integrate with apps like WhatsApp and Telegram. It learns your preferences over time through persistent memory.

Why should I host OpenClaw in the cloud instead of running it locally?

When you run OpenClaw locally, it stops when you close your laptop. Cloud hosting means your AI assistant runs 24/7, can receive messages while you sleep, execute scheduled tasks, and be accessed from any device. Clowdbot handles all the server setup so you can focus on using your assistant.

How much does it cost to run OpenClaw 24/7?

Clowdbot uses true pay-as-you-go pricing. You pay $0.50 to launch an instance, then only pay for the LLM tokens you actually use (at competitive rates). There are no hourly compute fees, no subscriptions, and no minimums. If your assistant is idle, you pay nothing beyond the initial launch fee.

Can I use my own API keys, or do I have to use Clowdbot's LLM access?

Clowdbot provides built-in access to Claude, GPT, Gemini, and other LLMs at competitive rates with no API key management required. This is the easiest option for most users. If you prefer to use your own API keys, you can configure that in your OpenClaw instance settings.

Is my data secure on a cloud-hosted OpenClaw instance?

Yes. Each Clowdbot instance is isolated and runs in its own secure environment. Your persistent memory, skills, and connected accounts are stored encrypted and only accessible to you. We don't have access to your conversations or data.

What's the difference between Clowdbot and other AI hosting services?

Clowdbot is specifically built for OpenClaw (Clawdbot). Unlike generic cloud VMs, we handle all the dependencies, configuration, and LLM integrations automatically. And unlike AI API services, you get a full persistent AI assistant with memory and skills, not just API access.

What is ATXP?

ATXP stands for Agent Transaction Protocol. It's the magic behind Clowdbot's true pay-per-use pricing‚ÄîATXP handles billing, LLM access, and infrastructure so you get all major LLMs and tons of built-in tools without juggling API keys or subscriptions. Just fund your account and go.
